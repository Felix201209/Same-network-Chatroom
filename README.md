Follow the steps below to run the project:

```bash
# Clone the repository (optional)
```

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

\. "$HOME/.nvm/nvm.sh"

# Download and install Node.js:
```
