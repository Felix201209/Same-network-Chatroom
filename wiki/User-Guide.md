## Troubleshooting

### Can't Send Messages

**Problem**: Messages not sending

**Solutions**:
1. Check if you're muted (notification will tell you)
2. For non-friends, wait for reply to first message
3. Check internet connection
4. Refresh the page

### Can't See Messages

**Problem**: Messages not appearing

**Solutions**:
1. Check internet connection
2. Refresh the page
3. Log out and log back in
4. Clear browser cache

### File Upload Failed

**Problem**: Can't upload files

**Solutions**:
1. Check file size (max 100MB)
2. Check internet connection
3. Try a smaller file
4. Try different file format

### Can't Login

**Problem**: Login not working

**Solutions**:
1. Check username and password
2. Ensure caps lock is off
3. Check if account is banned (message will show)
4. Contact administrator

### Disconnected

**Problem**: Connection lost

**Solutions**:
1. Check internet connection
2. Refresh the page
3. Application will auto-reconnect
4. Log out and back in if persists

### Slow Performance

**Problem**: Application is slow

**Solutions**:
1. Close other browser tabs
2. Clear browser cache
3. Check network speed
4. Contact administrator
